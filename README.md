# linter-ui-minimal

Lightweight front of [linter](https://github.com/steelbrain/linter) package. Status & linter panels on board.

![demo](https://github.com/asiloisad/pulsar-linter-ui-minimal/blob/master/assets/demo.png?raw=true)

## Commands

In `atom-workspace` there are available commands:

- `linter-ui:toggle`: (default `Alt-L`) show/hide linter panel.

In `atom-text-editor:not([mini])` there are available commands:

- `linter-ui:inspect`: inspect issue at cursor position if exists. Available in context-menu.
- `linter-ui:next`: (default `Alt-;`) scroll-to and inspect next issue if exists.
- `linter-ui:prev`: (default `Alt-'`) scroll-to and inspect previous issue if exists.

## Installation

To install `linter-ui-minimal` search for [linter-ui-minimal](https://web.pulsar-edit.dev/packages/linter-ui-minimal) in the Install pane of the Pulsar settings or run `ppm install linter-ui-minimal`. Alternatively, you can run `ppm install asiloisad/pulsar-linter-ui-minimal` to install a package directly from the GitHub repository.
